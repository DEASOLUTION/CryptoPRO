# CryptoPRO


Скрипт автоматически находит все контейнеры КриптоПро на съемных носителях (флешках, USB-токенах) и устанавливает в них сертификаты, которые расположены на тех же съемных носителях.

Предварительные требования
Установлен КриптоПро CSP.

Съемные носители с контейнерами закрытых ключей подключены к компьютеру.

На тех же съемных носителях находятся файлы сертификатов (с расширениями .cer, .crt, .p7b, .pfx).

Порядок использования
Подготовка:

Подключите все съемные носители с контейнерами и сертификатами к компьютеру.

Убедитесь, что КриптоПро CSP корректно видит контейнеры (можно проверить через КриптоПро PKI).

Запуск скрипта:

Скопируйте файлы auto_install_cert.bat и auto_install_cert.ps1 в одну папку.

Запустите файл auto_install_cert.bat от имени администратора (это важно для работы с КриптоПро).

Наблюдение за процессом:

Скрипт выведет список найденных контейнеров на съемных носителях.

Для каждого контейнера скрипт определит диск и найдет на нем сертификаты.

Каждый найденный сертификат будет установлен в контейнер с отображением результата (успех или ошибка).
